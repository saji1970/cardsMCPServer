# cardsMCPServer

MCP server and HTTP admin UI live in **`mcp-server/`**.

## Railway (Railpack)

Deploy from this **repo root** (do not set a subdirectory unless you know you want `mcp-server` only). Root **`package.json`** installs and builds **`mcp-server`**, then starts **`npm run start:http`**.

See **`mcp-server/RAILWAY.md`** for env vars and local dev.
